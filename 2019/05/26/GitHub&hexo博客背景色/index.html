通过网络查询的结果
背景色：“#f1fafa” 适合做正文的背景色，比较淡雅。配以同 色系的蓝色、深灰色或黑色文字都很好。

通过搜索Background 找到2个文件_base.scss和_variables.scss

在D:\Blog\themes\even\source\css这个文件夹

修改_variables.scss文件中的如下内容为：

// Background color of the site.
$global-background: #f1fafa !default;
