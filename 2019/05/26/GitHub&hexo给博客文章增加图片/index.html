First 相对引用

由于首页和主页相对图片位置不一致，网站首页和内页很可能仅有一处能正常加载。

1 调整程序配置文件_config.yml 里的post_asset_folder:这个选项设置为True，如果没有则忽略

2 在source下创建 img文件夹，先把图片*.jpg/png复制到这个文件夹

3 最后在.md中想引入图片.jpg时，只需要在*.md中按照markdown的格式引入图片：
