#需要用的软件：
node.js，git，Emeditor

#需要用到的网络工具
http://laobubu.net/HyperMD/#./docs/zh-CN/README.md
前期的准备工作略过，

【D:\Blog\source_posts】 这个是本地的文章保存目录，

  • 1 利用emeditor工具将该目录中的文件进行复制编辑并保存
  • 2 我们也可以直接复制网络上的文章，到上面的网站进行源码转换，转换完成之后，复制进emeditor进行保存即可！
  • 3 在Blog根目录，右键执行 git bash here，执行命令 hexo s 实时查看修改成果

    常用维护管理命令

清理缓存

hexo clean

重新生成

hexo generate

启动 server

hexo server -d

部署

hexo deploy

在BLOG文件夹右键选择git bash here，输入部署并发布命令
hexo d -g
